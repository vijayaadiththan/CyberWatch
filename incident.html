info Incident Overview

WannaCry is a ransomware cryptoworm that targeted computers running the Microsoft Windows operating system by encrypting data and demanding ransom payments in the Bitcoin cryptocurrency. The attack utilized the EternalBlue exploit, developed by the NSA and subsequently leaked by the Shadow Brokers group, to propagate through networks by exploiting a vulnerability in the Server Message Block (SMB) protocol.
