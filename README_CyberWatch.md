# CyberWatch

CyberWatch is a cybersecurity incident intelligence and analytics platform developed to help users explore, analyze, and learn from major cyber security incidents.

## Features
- Dynamic Incident Library
- Detailed Cyber Attack Analysis Pages
- Related Incident Recommendations
- Cyber Threat Analytics Dashboard
- JSON-Based Incident Database

## Technologies Used
- HTML5
- CSS3
- JavaScript
- JSON
- Tailwind CSS
