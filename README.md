# CyberWatch

CyberWatch is a cybersecurity incident intelligence and analytics platform developed to help users explore, analyze, and learn from major cyber security incidents. The platform provides detailed incident case studies, attack analysis, and visual threat intelligence dashboards through an interactive web interface.

## Features

* Dynamic Incident Library
* Detailed Cyber Attack Analysis Pages
* Related Incident Recommendations
* Cyber Threat Analytics Dashboard
* Incident Categorization and Severity Classification
* Timeline-Based Incident Exploration
* Responsive Modern UI
* JSON-Based Incident Database

## Technologies Used

* HTML5
* CSS3
* JavaScript
* JSON
* Tailwind CSS

## Key Modules

### Homepage

Provides an overview of the CyberWatch platform and highlights major cyber incidents.

### Incident Library

Displays a collection of significant cyber attacks with filtering and exploration capabilities.

### Incident Details

Provides detailed information about individual incidents including:

* Overview
* Timeline
* Impact Analysis
* Systems Affected
* Mitigation Strategies
* Related Incidents

### Analytics Dashboard

Visualizes cybersecurity trends and insights including:

* Threat Distribution
* Severity Analysis
* Incident Categories
* Global Impact Insights
* Attack Vector Analysis

## Dataset

The project contains a curated collection of major cybersecurity incidents including:

* WannaCry
* Equifax Data Breach
* SolarWinds Supply Chain Attack
* Colonial Pipeline Attack
* MOVEit Transfer Breach
* MGM Resorts Cyberattack
* Stuxnet
* Target Data Breach
* NotPetya
* Log4Shell

## Purpose

This project was developed as an educational cybersecurity learning platform to improve understanding of real-world cyber attacks, their impact, and defensive strategies.
